# 🐍 Python Programs Collection

This repository contains simple Python programs for beginners.  
It covers basic concepts like loops, patterns, lists, and logic building.

## 📂 Programs Included

- 🔢 Finding odd numbers
- ✖️ Multiplication table
- 🔍 Search in list
- 🔺 Triangle programs
- 🎨 Pattern printing
- 🔼 Pyramid printing

## ▶️ How to Run

1. Install Python
2. Download or clone this repo
3. Run any file:

```bash
python filename.py
```
